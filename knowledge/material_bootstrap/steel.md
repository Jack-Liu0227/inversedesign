# Steel Design Bootstrap

## Typical Objective Patterns
- Simultaneous YS/UTS improvement with acceptable elongation.
- Strength-ductility tradeoff control under realistic process constraints.

## Composition Guidance
- Fe remains base element.
- Ni/Co/Mo/Cr adjustments should be conservative and metallurgically plausible.
- Avoid composition combinations that imply unrealistic phase behavior without process support.

## Heat Treatment Method Guidance
- Keep one full route in a single sentence.
- Include key steps such as homogenization/hot rolling/aging when needed.
- Route should be executable, not fragmented keywords.

## Candidate Strategy
- Reuse high-score valid patterns from previous round.
- If risk tags concentrate in brittleness or process infeasibility, reduce aggressive strengthening changes.
